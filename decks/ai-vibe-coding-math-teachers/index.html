WEB LLM VS WORKSPACE

대화창에서 작업공간으로 이동해야 할 때가 있습니다.

WEB LLM

묻고 답하기

  • 아이디어 정리
  • 문서 요약
  • 수업자료 초안
  • 빠른 질의응답
IDE / AGENT

만들고 고치기

  • 파일 생성과 수정
  • 코드 실행
  • 오류 확인
  • 결과물 검증
08 / 19
WHEN TO USE

답변이 필요하면 웹 LLM,
결과물이 필요하면 작업환경.

웹 LLM이 적합한 경우

빠른 사고 보조

  • 아이디어 발산
  • 문장 다듬기
  • 간단한 자료 초안
작업환경이 필요한 경우

반복 제작과 검증

  • 여러 파일이 필요한 작업
  • 실행과 수정이 반복되는 작업
  • 나만의 도구 제작

작은 실험부터 시작한다
